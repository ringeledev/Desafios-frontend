# Desafios-frontend
Desarrollando desafíos de la comunidad, para crecer en el área de desarrollo de software.

- `index.html`: Estructura del desafío.
- `styles.css`: Estilos aplicados.
- Imágenes u otros recursos necesarios.

- Visual Studio Code
